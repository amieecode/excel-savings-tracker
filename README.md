# Monthly Savings Tracker 
## Project Overview

The Monthly Savings Tracker is an Excel-based tool designed to help users plan, track, and monitor their savings throughout the year. It allows users to:
- Set a yearly savings goal
- Break it down into monthly targets
- Track actual savings vs target
- Calculate differences and remaining balances
- Dynamically adjust monthly targets either manually or automatically
This tracker provides a flexible, user-friendly approach to personal finance management.

## Features
1. Year Selection
-- Select the year for which you are tracking savings.
2. Dynamic Month Selection
-- Choose a month from a dropdown to view or edit data for that month.
3. Yearly & Monthly Target Management
- Enter a yearly savings goal.
- Monthly targets can either be calculated automatically (Yearly Target ÷ 12) or entered manually using a toggle switch.
4. Data Table
Track:
- Date money came in
- Total income received
- Monthly target
- Actual amount saved
- Difference (Actual – Target)
- Running balance against yearly goal
5. Automatic Calculations
- Total saved so far
- Remaining target
- Difference per month
- Balance left for the year
6. Conditional Formatting & Highlights
- Highlight the selected month for clarity
- Visual cues for oversaving or undersaving
7. Flexible Inputs
- Manual or automatic monthly target entry
- Real-time updates for all calculations

## How to Use
1. Set the Year
- Enter the year in the control panel.
2. Set Your Yearly Savings Target
- Input your goal amount (e.g., 120,000).
3. Monthly Target Mode
- Automatic  to divide the yearly target by 12
4. Select
- Use the dropdown to pick a month and view or edit that month’s data.
5. Enter Actual Savings
- Input the amount you saved for the selected month.
6. Review Dashboard
- Monitor total savings, remaining balance, and monthly differences.
